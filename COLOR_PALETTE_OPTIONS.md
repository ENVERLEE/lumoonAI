# 색상 팔레트 후보

## 현재 팔레트 (ChatGPT Style)
- **Accent**: #10a37f (에메랄드 그린)
- **스타일**: 모노크롬 + 한 가지 accent 색상
- **느낌**: 깔끔하고 전문적
